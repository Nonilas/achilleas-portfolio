# Art Assets Research: Byzantine Scholar's Study Room

**Date:** 2026-02-24
**Purpose:** Find free medieval/fantasy game art assets for a 2.5D parallax study room scene in a portfolio website.

---

## Summary

This document catalogues free (or very low cost) game art assets suitable for composing a Byzantine scholar's study room with a 2.5D parallax effect. Assets are organized by layer/role in the scene. Where good free standalone assets are unavailable, the recommendation is to fall back to CSS/SVG placeholders (already being built by another agent).

## 6. Recommended Asset Combination for the Study Room Scene

Based on the research, here is the recommended free asset combination for the 2.5D parallax Byzantine study room:

### Layer 1 - Far Background (Stone Walls / Windows)
**Primary:** Free Castle Interior Pixel Backgrounds (CraftPix) -- use the "Stained Glass Halls" or "Castle Interior" scene as base. Separate PSD layers for parallax.

### Layer 2 - Mid Background (Bookshelves)
**Primary:** The Library Indoor Asset Pack (sparksandpixels) -- bookshelves and decorative elements.
**Supplement:** Pixel Art - Library of Books (Arijkx) -- additional book/shelf variety.

### Layer 3 - Desk / Furniture
**Primary:** Compose from The Library Indoor Asset Pack tables + individual object sprites.

### Layer 4 - Character
**Primary:** Free NPC Mage by CreativeKind (best for stationary scholar).
**Alternative:** Wizard Pack by LuizMelo (CC0, use idle frames).

### Layer 5 - Foreground Objects (on desk)
- **Candle:** Candle Sprite from OpenGameArt (CC0)
- **Globe/Astrolabe:** Pixel Art Globe from OpenGameArt (CC0)
- **Books/Scrolls:** 496 Pixel Art Icons pack from OpenGameArt (Public Domain)
- **Compass:** Pixel Art Compass from OpenGameArt

## 7. Fallback Strategy

If the free assets above do not achieve the desired visual quality or stylistic consistency for the Byzantine aesthetic, the scene should use **CSS/SVG placeholders** which are already being developed by another agent. The CSS/SVG approach offers:

- Complete control over the Byzantine visual style (gold, deep purple, stone textures)
- Perfect resolution independence (vector-based)
- Lighter page weight (no large image downloads)
- Easier parallax implementation with CSS transforms
- Consistent art style across all elements

The pixel art assets listed above can serve as **visual reference** for the CSS/SVG implementation even if not used directly.
